Objeto Window

Un Objeto para gobernarlos a todos. Un Objeto para encontrarlos, un Objeto para atraerlos a todos y atarlos en el código.

Introducción

Cada variable, objeto y función creada en Javascript es parte de un Objeto Global, en el caso de los navegadores es el objeto Window.

El objeto window representa la ventana o pestaña que contiene el DOM, que es la estructura de nodos que tiene la página web.

Las versiones 3.0 de los navegadores Internet Explorer y Netscape Navigator introdujeron el concepto de Browser Object Model o BOM, que permite acceder y modificar las propiedades de las ventanas del propio navegador.

Pero algunas de estas opciones están desactivadas en los navegadores por motivos de seguridad...

El mayor inconveniente de BOM es que, al contrario de lo que sucede con DOM, ninguna entidad se encarga de estandarizarlo o definir unos mínimos de interoperabilidad entre navegadores.

Algunos de los elementos que forman el BOM son los siguientes:

  • Crear, mover, redimensionar y cerrar ventanas de navegador.
  • Obtener información sobre el propio navegador.
  • Propiedades de la página actual y de la pantalla del usuario.
  • Gestión de cookies.
  • Objetos ActiveX en Internet Explorer.

Además en un navegador con pestañas, como Firefox, cada pestaña contiene su propio Objeto Window.

OBJETO WINDOW

El objeto Window se considera el objeto más importante del lenguaje JavaScript porque a partir de él, se pueden gestionar las ventanas del navegador utilizando una serie de propiedades y métodos.

Características

  • Se considera como un objeto implícito.
  • Soportado por todos los navegadores
          
                  function myFunction() {
                      window.document.bgColor = "yellow";
                  }
                  function myFunction2() {
                     document.bgColor = "white";
                  }

document

Regresa el objeto "document" para la ventana

Cuando un documento HTML es cargado en un navegador, este se convierte en un objeto document. El objeto document es el nodo raíz del documento HTML y del que derivan los demás.

El objeto document es parte del objeto Window y se puede acceder a él mediante window.document.

frameElement

Devuelve el elemento <iframe> de la ventana actual, si no es un iframe devuelve null


function myFunction() {
    var frame = window.frameElement;
    if (frame) frame.src = "http://www.w3schools.com/";
}

Es el conjunto de marcos de la ventana, la sintaxis es parecida a un array.

Screen

Devuelve el objeto Screen para la pantalla que contiene información sobre la ventana del usuario.

  • availHeight: Devuelve la altura de la pantalla sin incluir la barra del navegador
  • availWidth: Devuelve la anchura de la pantalla sin incluir la barra del navegador
  • colorDepth: Devuelve la profundidad de color de la pantalla para enseñar imágenes
  • height: Devuelva la altura total de la pantalla
  • pixelDepth: Devuelve la resolución de la pantalla en pixel por bits
  • width: Devuelva la anchura total de la pantalla

Atob y Btoa

atob() decodifica un string codificado en base-64

y btoa() lo codifica


function myFunction() {
    var str = "Hello World!";
    var enc = window.btoa(str);
    var dec = window.atob(enc);

    var res = "Encoded String: " + enc + "
" + "Decoded String: " + dec;
    document.getElementById("demo").innerHTML = res;
}

Blur y Focus

Blur quita el focus de la ventana actual y Focus asigna el focus a la ventana actual.


function windowBlurFocus(){
  var focused = true;
  window.onfocus = function() {
    focused = true;
    onFocus();
  };
  window.onblur = function() {
    focused = false;
    onBlur();
  };
}

resizeBy()

Redimensiona ventana navegador a un tamaño específico
indicando cantidad relativa respecto tamaño actual.
Sintaxis : window.resizeBy(width,height)
Nueva anchura : (anchura_anterior + x)
Nueva altura : (altura_anterior + y)
Valores parámetros:
width - Número positivo / negativo especifica cantidad pixeles se redimensiona ancho pagina. (Requerido)
height - Número positivo / negativo especifica cantidad pixeles se redimensiona alto página (Requerido)

resizeTo()

Redimensiona tamaño navegador mediante ancho "x" , alto "y" específico.
No permiten valores negativos.
Sintaxis: window.resizeTo(width,height)
Valores parámetros:
width: Número pixeles tiene ancho ventana (Requerido)
height: Número pixeles tiene alto ventana (Requerido)
Valor Retorno : Ninguno.

scrollBy()

Desplazamiento scroll número específico píxeles.
Sintaxis : window.scrollBy(num_x , num_y)
Valores parámetros :
num_x : Numero píxeles se desplaza en eje x (Requerido)
num_y : Numero pixeles se desplaza en eje y (Requerido)
Valor Retorno : Ninguno.

scrollTo()

Desplazamiento del documento especificado mediante coordenadas "x" horizontal , "y" vertical.
Sintaxis : window.scrollTo(posicion_x , posicion_y)
Valores parametros:
posición x : Coordenadas desplaza eje "x" . (Requerido)
posición y : Coordenadas desplaza eje "y" . (Requerido)
Valor Retorno : Ninguno.

stop()

Función que detiene la carga de la ventana.
Es un método útil para cargar imagenes o frames que exceden demasiado tiempo.
Sintaxis : window.stop()
Valores parámetros : Ninguno.
Valor de retorno : Ninguno.
