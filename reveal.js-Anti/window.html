Objeto Window

Un Objeto para gobernarlos a todos. Un Objeto para encontrarlos, un Objeto para atraerlos a todos y atarlos en el código.

Introducción

Cada variable, objeto y función creada en Javascript es parte de un Objeto Global, en el caso de los navegadores es el objeto Window.

El objeto window representa la ventana o pestaña que contiene el DOM, que es la estructura de nodos que tiene la página web.

Las versiones 3.0 de los navegadores Internet Explorer y Netscape Navigator introdujeron el concepto de Browser Object Model o BOM, que permite acceder y modificar las propiedades de las ventanas del propio navegador.

Pero algunas de estas opciones están desactivadas en los navegadores por motivos de seguridad...

El mayor inconveniente de BOM es que, al contrario de lo que sucede con DOM, ninguna entidad se encarga de estandarizarlo o definir unos mínimos de interoperabilidad entre navegadores.

Algunos de los elementos que forman el BOM son los siguientes:

  • Crear, mover, redimensionar y cerrar ventanas de navegador.
  • Obtener información sobre el propio navegador.
  • Propiedades de la página actual y de la pantalla del usuario.
  • Gestión de cookies.
  • Objetos ActiveX en Internet Explorer.

Además en un navegador con pestañas, como Firefox, cada pestaña contiene su propio Objeto Window.

OBJETO WINDOW

El objeto Window se considera el objeto más importante del lenguaje JavaScript porque a partir de él, se pueden gestionar las ventanas del navegador utilizando una serie de propiedades y métodos.

Características

  • Se considera como un objeto implícito.
  • Soportado por todos los navegadores

PROPIEDADES

blalblalbalblalablalblagbla

Existen propiedades que ya no se utilizan o que su uso está bloqueado como hemos visto.

closed

Devuelve valor "booleano" indicando cuando ventana ha sido cerrada o no

defaultStatus

Ajusta o regresa el texto por defecto dentro barra de estado de una ventana ( Deprecated )

document

Regresa el objeto "document" para la ventana

frameElement

Regresa el elemento <iframe> la cual es la ventana actual que está ingresada

frames

Conjunto de marcos de la ventana

history

Conjunto de elementos que representan las URL visitadas

innerHeight

Altura utilizable de la ventana.

innerWidth

Ancho utilizable de la ventana

length

Regresa numero elementos <iframe> en la ventana actual

localStorage

Regresa una referencia al objeto localStorage usado en
el almacenamiento de datos , este objeto similar a sessionStorage , la diferencia es que mientras los datos almacenados en localStorage no tiene fecha de expiración , datos almacenados en sessionStorage son eliminados cuando la sesión de navegacion finaliza "cuando el usuario cierra la ventana del navegador"

location

Regresa el objeto Location para la ventana del navegador

name

Fija o regresa el nombre de la ventana.

navigator

Regresa una referencia al objeto "navigator el cual puede ser utilizado para obtener información sobre la aplicacion que está ejecutando el script

opener

Regresa una referencia de la ventana que creo la ventana actual

outerHeight

Regresa el alto de todos los navegadores , incluyendo la barra de notificaciones (Si se encuentra) y los bordes

outerWidth

Obtiene alto en pixeles de la ventana del navegador por completo
y representa el alto de todo el navegador , incluyendo la barra de notificaciones si se encuentra y los bordes

sessionStorage

Devuelve una referencia al objeto de almacenamiento local (local storage object) usando para almacenar información.
Solo guarda información para una sesión y se pierde cuando la ventana o pestaña es cerrada.

Screen

Devuelve el objeto Screen para la pantalla que contiene información sobre la ventana del usuario.

  • availHeight: Devuelve la altura de la pantalla sin incluir la barra del navegador
  • availWidth: Devuelve la anchura de la pantalla sin incluir la barra del navegador
  • colorDepth: Devuelve la profundidad de color de la pantalla para enseñar imágenes
  • height: Devuelva la altura total de la pantalla
  • pixelDepth: Devuelve la resolución de la pantalla en pixel por bits
  • width: Devuelva la anchura total de la pantalla

Atob y Btoa

atob() decodifica un string codificado en base-64

y btoa() lo codifica


function myFunction() {
    var str = "Hello World!";
    var enc = window.btoa(str);
    var dec = window.atob(enc);

    var res = "Encoded String: " + enc + "
" + "Decoded String: " + dec;
    document.getElementById("demo").innerHTML = res;
}

Blur y Focus

Blur quita el focus de la ventana actual y Focus signa el focus a la ventana actual.


function windowBlurFocus(){
  var focused = true;
  window.onfocus = function() {
    focused = true;
    onFocus();
  };
  window.onblur = function() {
    focused = false;
    onBlur();
  };
}
